02 April 2019

PsyCommando updated:

  • Added the improvised bolt action 7.62 caliber rifle from UristMcStation.

ingles98 updated:

  • You are now able to add a title to canvases and sign it if you so wish.
  • You are now able to edit the canvas as it will load the current picture state into the editor.
  • Canvas pixel width and height should have been 30, and not 31 as it was.
  • Currently working on a data bridge between the browser and DM which will substantially boost perfomance while saving the icon, enabling bigger icon sizes in the future.

ourpar updated:

  • 5 new tailoring machines, each with their own specific set of designs, you screwdriver the standard circuit printed from RD to access subtypes
  • Stethoscope and defibs can now be printed from the protolathe
  • Cheap insulated gloves now have a static 0.2 shock absorbtion, you probably shoudn't trust your life with them still
  • Restored any armor values caused by the recent autotailor PR
  • Cheap insulated gloves were reverted to have RNG based shock absorption again

psycommando updated:

  • Added bomb suit kits to security and science categories on the supply app.
  • Added eod shoes, bomb resistant shoes to go along with the bomb suits
